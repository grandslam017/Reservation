The Grand Slam Tennis Court ("เรา และ/หรือ บริษัท") ให้ความสำคัญกับการคุ้มครองข้อมูลส่วนบุคคลของผู้ใช้งาน และดำเนินการเก็บรวบรวม ใช้ และเปิดเผยข้อมูลส่วนบุคคลตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA) โดยมีรายละเอียดดังต่อไปนี้

1. ข้อมูลส่วนบุคคลที่เราเก็บรวบรวม

เราอาจเก็บรวบรวมข้อมูลส่วนบุคคลของท่านเมื่อสมัครสมาชิกหรือใช้งานแอปพลิเคชัน ดังนี้

1.1 ข้อมูลบัญชีผู้ใช้งาน

  • ชื่อ-นามสกุล
  • อีเมล
  • หมายเลขโทรศัพท์
  • รหัสผ่าน (จัดเก็บในรูปแบบที่มีการป้องกันตามมาตรฐานความปลอดภัย)
  • รูปภาพโปรไฟล์ (หากมี)
  • LINE User ID หรือข้อมูลที่ได้รับจากการเข้าสู่ระบบด้วย LINE (หากท่านเลือกใช้งาน)
  • ทะเบียนรถ

1.2 ข้อมูลการใช้งาน

  • ประวัติการจองสนามเทนนิส
  • ประวัติการทำรายการ
  • วันและเวลาที่เข้าใช้งาน
  • ข้อมูลการใช้งานระบบและบันทึกเหตุการณ์ (Log) ที่จำเป็นต่อการให้บริการและรักษาความปลอดภัยของระบบ

2. วัตถุประสงค์และฐานทางกฎหมายในการประมวลผลข้อมูล

เราเก็บรวบรวม ใช้ และประมวลผลข้อมูลส่วนบุคคลของท่านเพื่อวัตถุประสงค์ดังต่อไปนี้

2.1 เพื่อสมัครสมาชิก ยืนยันตัวตน และให้บริการจองสนามเทนนิส

ฐานกฎหมาย: การปฏิบัติตามสัญญา

2.2 เพื่อให้บริการ ดูแล และปรับปรุงประสิทธิภาพของแอปพลิเคชัน

ฐานกฎหมาย: ประโยชน์โดยชอบด้วยกฎหมายของบริษัท

2.3 เพื่อแจ้งเตือน ติดต่อสื่อสาร และให้บริการหลังการขาย

ฐานกฎหมาย: การปฏิบัติตามสัญญา

2.4 เพื่อรักษาความปลอดภัยของระบบ ป้องกันการทุจริต และตรวจสอบการใช้งานที่ผิดปกติ

ฐานกฎหมาย: ประโยชน์โดยชอบด้วยกฎหมาย

2.5 เพื่อส่งข่าวสาร โปรโมชั่น หรือกิจกรรมทางการตลาด (หากมี)

ฐานกฎหมาย: ความยินยอมของเจ้าของข้อมูล

3. การเปิดเผยข้อมูลส่วนบุคคล

เราจะไม่ขาย แลกเปลี่ยน หรือเปิดเผยข้อมูลส่วนบุคคลของท่านแก่บุคคลภายนอก เว้นแต่ในกรณีดังต่อไปนี้

  • ได้รับความยินยอมจากท่านในกรณีที่กฎหมายกำหนด
  • เป็นการปฏิบัติตามกฎหมาย คำสั่งศาล หรือคำสั่งของหน่วยงานราชการที่มีอำนาจ
  • เปิดเผยแก่ผู้ให้บริการที่เกี่ยวข้องกับการให้บริการของเรา เช่น ผู้ให้บริการระบบ Cloud ผู้ให้บริการระบบฐานข้อมูล หรือผู้ให้บริการด้านเทคโนโลยีสารสนเทศ ซึ่งจะดำเนินการภายใต้ข้อตกลงรักษาความลับและมาตรการคุ้มครองข้อมูลส่วนบุคคลตามกฎหมาย

ทั้งนี้ ข้อมูลส่วนบุคคลของท่านอาจถูกโอนไปยังต่างประเทศในกรณีที่ผู้ให้บริการระบบหรือผู้ประมวลผลข้อมูลตั้งอยู่ในต่างประเทศ โดยเราจะดำเนินการให้มีมาตรการคุ้มครองข้อมูลส่วนบุคคลตามที่กฎหมายกำหนด

4. ระยะเวลาการจัดเก็บข้อมูล

เราจะเก็บรักษาข้อมูลส่วนบุคคลของท่านตลอดระยะเวลาที่ท่านเป็นผู้ใช้งานแอปพลิเคชัน

เมื่อท่านลบบัญชีผู้ใช้งาน (Delete Account) หรือยกเลิกการใช้บริการ เราจะลบหรือทำลายข้อมูลส่วนบุคคลภายใน 90 วัน เว้นแต่กฎหมายกำหนดให้ต้องเก็บรักษาข้อมูลไว้เป็นระยะเวลานานกว่านั้น หรือมีความจำเป็นเพื่อการใช้สิทธิเรียกร้องตามกฎหมาย

5. สิทธิของเจ้าของข้อมูลส่วนบุคคล

ท่านมีสิทธิตามกฎหมาย PDPA ดังต่อไปนี้

  • สิทธิในการขอเข้าถึงข้อมูลส่วนบุคคลและขอรับสำเนาข้อมูล
  • สิทธิในการขอแก้ไขข้อมูลให้ถูกต้อง เป็นปัจจุบัน และสมบูรณ์
  • สิทธิในการขอรับหรือโอนย้ายข้อมูลส่วนบุคคล (Data Portability) ตามที่กฎหมายกำหนด
  • สิทธิในการขอให้ลบ ทำลาย หรือทำให้ข้อมูลไม่สามารถระบุตัวบุคคลได้
  • สิทธิในการขอระงับการใช้ข้อมูลส่วนบุคคล
  • สิทธิในการคัดค้านการประมวลผลข้อมูลในกรณีที่กฎหมายกำหนด
  • สิทธิในการถอนความยินยอมเมื่อใดก็ได้ โดยการถอนความยินยอมจะไม่กระทบต่อการประมวลผลข้อมูลที่ได้ดำเนินการไปก่อนหน้า
  • สิทธิในการร้องเรียนต่อสำนักงานคณะกรรมการคุ้มครองข้อมูลส่วนบุคคล หากเห็นว่าการประมวลผลข้อมูลของเราไม่เป็นไปตามกฎหมาย

6. ผลกระทับหากไม่ให้ข้อมูลส่วนบุคคล

หากท่านไม่ให้ข้อมูลส่วนบุคคลที่จำเป็นสำหรับการสมัครสมาชิกหรือการใช้บริการ เราอาจไม่สามารถให้บริการบางส่วนหรือทั้งหมดแก่ท่านได้

7. มาตรการรักษาความมั่นคงปลอดภัยของข้อมูล

เราใช้มาตรการด้านเทคนิคและการบริหารจัดการที่เหมาะสมในการรักษาความมั่นคงปลอดภัยของข้อมูลส่วนบุคคล เพื่อป้องกันการสูญหาย การเข้าถึง การใช้ การเปลี่ยนแปลง การเปิดเผย หรือการทำลายข้อมูลโดยไม่ได้รับอนุญาต รวมถึงการกำหนดสิทธิ์การเข้าถึงข้อมูลเฉพาะผู้ที่มีหน้าที่เกี่ยวข้อง

8. การแก้ไขนโยบายความเป็นส่วนตัว

เราอาจปรับปรุงหรือแก้ไขนโยบายความเป็นส่วนตัวนี้เป็นครั้งคราว เพื่อให้สอดคล้องกับกฎหมายหรือการเปลี่ยนแปลงในการให้บริการ โดยจะแจ้งให้ผู้ใช้งานทราบผ่านแอปพลิเคชันหรือช่องทางที่เหมาะสม

9. ช่องทางการติดต่อ

หากท่านมีข้อสงสัยเกี่ยวกับนโยบายความเป็นส่วนตัวนี้ หรือต้องการใช้สิทธิของเจ้าของข้อมูลส่วนบุคคล สามารถติดต่อผู้ควบคุมข้อมูลส่วนบุคคลได้ที่
